Sticky Notes in Menubar

Quick Note will let you take notes quickly without asking for file name and location.
You can also open a new note with a shortcut. Create notes with beautiful background and support for Markdown syntax.

Free to download on the Mac app store

Watch demo

Features

New note from menu bar

Create new note from menu bar with ease. A new note opens up without asking location or filename

A shortcut can be assigned to new note from Preferences.

Expand. Collapse.

You can collapse the note when you work on other app. Collapsing keeps the note floating but does not occupy much space
